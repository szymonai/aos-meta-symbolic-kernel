## Non-Redistribution

The public repository must not contain:

- patient records
- DICOM files
- NIfTI files
- masks or segmentations
- medical images
- model checkpoints
- local dataset paths
- internal audit logs
- controlled technical artifacts

The public evidence is intentionally limited to aggregate metadata and
defensive claim-boundary flags.
